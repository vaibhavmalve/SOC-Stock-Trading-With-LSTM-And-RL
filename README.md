# Stock Trading With LSTM and Reinforcement Learning  
### Seasons of Code 2026 — WnCC IIT Bombay

A guided mentorship project focused on exploring stock market prediction using Machine Learning and Reinforcement Learning.

Over 7 weeks, we’ll go from Python basics and technical analysis to implementing LSTM and RL-based trading models on real market data.

---

### Week 1
- Python Basics

## Basic GitHub Workflow

During the project, we’ll be using Git and GitHub for collaboration, version control, and maintaining the repository.

### 1. Fork the Repository
Create your own copy of the project repository using the **Fork** button on GitHub.

---

### 2. Clone the Repository

```bash
git clone <repository-link>
```

---

### 3. Move Into the Project Folder

```bash
cd <project-folder>
```

---

### 4. Add Upstream Remote

```bash
git remote add upstream <original-repository-link>
```

---

### 5. Create a New Branch

```bash
git checkout -b <branch-name>
```

---

### 6. Make Changes
Work on your assigned tasks and update the required files.

---

### 7. Check Modified Files

```bash
git status
```

---

### 8. Stage Changes

```bash
git add .
```

---

### 9. Commit Changes

```bash
git commit -m "Meaningful commit message"
```

---

### 10. Push Changes

```bash
git push origin <branch-name>
```

---

### 11. Create a Pull Request
Open a Pull Request (PR) on GitHub for review and merging.

---

### 12. Sync Your Fork

```bash
git checkout main
git pull upstream main
git push origin main
```
